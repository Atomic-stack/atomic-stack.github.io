And A Little CSS

Just enough to make things look nice.

But really, this is all about HTML and what you can do with Just HTML.

Like using a <head> tag for all the info about your page, like its title, language, behavior, and style. This stuff does not show up on the page, but it's still important for any website.

Then there's the <body> – that's where all the stuff you can actually see goes. Like this text.

We also made a few tweaks to make it look better:

  1. Centered the page
  2. Added some padding around it
  3. Set a nicer line length (about 70-characters)
  4. Increased the space between lines
  5. Made the font size a little bigger

And that's really all it takes to make a nice website. But we can do way more than that with just HTML.

Images

You can add a picture with the <img src="url"> tag. Also add a caption using the alt attribute while you're at it.

Formatting

<strong> is for bold text

<em> is for italic

<u> is for underline

You can also use <sup> to write stuff up here.

Or <sub> to put stuff down here.

You also have <pre> for preformatted blocks.
