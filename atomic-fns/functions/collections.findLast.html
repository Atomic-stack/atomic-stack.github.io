Function findLast

  • findLast<T>(arr: Iterable<T>, fn: Iteratee<T, any, any>): T | undefined

  • This method is like find except that it iterates from right to left.

    Example

    findLast([1, 2, 3, 4], (n) => n % 2 === 1)
    //  => 3

    Returns

    The matched element, else undefined.
